glio.js

Trigger anything you want when user try to go in the screen corners.
Go to the points in the document and see.

View on GitHub Download .zip Download .tar.gz

This library works with two arguments: 'direction' and a 'callback', who will be triggered. glio(direction, callback).


Usage:

  • glio.init(direction, callback);
    Use the method init to start. The callback just trigeer one time.

Directions:

  • glio.init("top-left", function () { alert("Now is Top-left."); });
  • glio.init("top-right", function () { alert("Now is Top-Right."); });
  • glio.init("bottom-left", function () { alert("Now is Bottom-Left"); });
  • glio.init("bottom-right", function () { alert("Now is Bottom-Right."); });
