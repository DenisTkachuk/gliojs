glio.js

Trigger anything you want when user try move out your site.

View on GitHub Download .zip Download .tar.gz

This library works with two arguments. The direction, who at moment is just "out" and a callback, who will be triggered: glio(direction, callback)


Usage:

  • glio.init("out", function () { alert("Hi."); });
    Use the method init to start. The callback just trigeer one time.
