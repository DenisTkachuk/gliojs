glio.js

A small Javascript library to trigger a callback when user try move out your site.

View on GitHub Download .zip Download .tar.gz

This library works with two arguments. The direction, who at moment is just "out" and a callback. See: glio(direction, callback)

Examples:

  • glio("out", function () { alert("Hi."); });
    Sets the color of the Text to the element.
