glio.js

Trigger anything you want when user try to go in the screen corners.
Go to the points in the document and see.

Star this Project Download .zip Download .tar.gz

The arguments are a Arrays, where the first index is the direction and the second is a callback. glio.init( [ direction, callback ], [ direction, callback ] );.


Usage:

  • glio.init( [ direction, callback ] );
    Use the method init to start. The callback just trigeer one time.

Directions:

  • top
  • top-left
  • top-right
  • bottom-left
  • bottom-right

Configurations:

    Set the configs before the init method. glio.config.key = value;
  • screenWidthFragment: the quantity of parts the height and width screen will be divided. Default: 12.
  • centerTopHeight: height of 'top' direction. When the mouse is equal or major this value, the callback is triggered. Default: 10.
  • heightTopLeft: height of 'top-left' direction. When the mouse is equal or major this value, the callback is triggered. Default: 30.
  • heightTopRight: height of 'top-right' direction. When the mouse is equal or major this value, the callback is triggered. Default: 30.
